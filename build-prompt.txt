The backend is already fully built in server/. Your ONLY job is to build the Next.js 14 frontend in a `client/` subfolder, then commit and push everything to GitHub.

DO NOT use '&&' in any shell commands you run — this is Windows PowerShell. Use ';' instead to chain commands.

## STEP 1 — Scaffold Next.js app
Run:
npx create-next-app@14 client --typescript --tailwind --eslint --app --src-dir --import-alias @/*
If it prompts interactively, answer Yes to all questions.

## STEP 2 — Install extra deps inside client/
cd client
npm install zustand axios recharts @hello-pangea/dnd lucide-react clsx @tailwindcss/forms @tailwindcss/typography

## STEP 6 — Build all source files

### client/src/lib/api.ts
Axios instance. baseURL = process.env.NEXT_PUBLIC_API_URL. Request interceptor: attach Authorization Bearer token from localStorage key 'orion_token'. Response interceptor: on 401, clear localStorage and redirect to /login.

### client/src/store/authStore.ts (Zustand)
State: user (id/name/email/role), token string.
login(email, password): POST /auth/login, store token in localStorage('orion_token') and state.
logout(): clear state and localStorage, window.location.href = '/login'.
On init: read token from localStorage.

### client/src/store/dashboardStore.ts (Zustand)
State: projects[], tasks[], agents[], loading boolean.
fetchProjects(): GET /projects.
fetchTasks(projectId?): GET /tasks with optional project_id param.
fetchAgents(): GET /agents.

## STEP 7 — Root package.json
Update scripts (use semicolons not &&, wrap in proper JSON):
{
  "scripts": {
    "dev": "concurrently \"npm run server\" \"npm run client\"",
    "server": "node server/index.js",
    "client": "cd client && npm run dev",
    "install:all": "npm install && cd server && npm install && cd ../client && npm install"
  }
}
Run: npm install concurrently (at root)
